飞享 IM Windows 隐私政策

生效日期：2026-07-09
适用版本：Windows 0.1.0
适用环境：当前公开分发版本

我们处理的信息

  • 账号信息：账号、昵称、用户 ID、头像、登录状态和客户端版本。
  • 登录凭证：业务 token、IM 登录凭证和云信运行时配置。客户端不得在普通日志中输出 token、nimtoken、密码或 app secret。
  • 通讯信息：会话列表、单聊、群聊、文本、自定义消息、图片、文件、语音消息和红包领取通知。
  • 联系人和群组：好友、群成员、群资料、群公告、群文件、群相册和群设置。
  • 钱包与红包：余额、账单、充值、提现、银行卡、支付密码状态、红包发送、领取、过期、退款和流水状态。
  • 内容与互动：签到、收藏、朋友圈、发现/探索入口、H5 页面访问状态和举报反馈。

第三方 SDK

客户端使用云信 IM SDK 提供即时通讯能力。云信 SDK 可能处理 IM 登录、消息收发、连接状态和 SDK 运行日志。客户端不得把云信 app secret 放入 Windows 包；运行所需 appkey 通过后端配置链路获取。

本地存储与安全

客户端通过统一存储抽象保存登录态和必要凭证。敏感字段不得写入普通日志、下载页或公开文档。账号密码、SSH 密码、签名密码、云信 secret 和后端 secret 不进入客户端仓库。

钱包和红包

钱包和红包涉及资金状态。客户端展示余额、账单、红包和交易状态，并通过后端接口完成校验。充值、提现、绑卡、支付密码、发红包、领红包、退款等行为以服务端结果为准。

H5 与外部链接

Windows 当前采用外部浏览器打开或安全阻断策略，不承诺内嵌 WebView2、JS bridge 或高权限 H5。客户端不得在 H5 URL 中拼接 token、nimtoken、密码或支付凭证。

不支持能力

Windows 客户端不迁移旧 Android 端电话拦截、通话记录、系统悬浮窗和应用内安装 APK 等高敏能力。
