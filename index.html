安装方式

  1. 下载 ZIP 和 SHA256，确认文件来自本页面。
  2. 校验 ZIP SHA256 与页面一致。
  3. 完整解压 ZIP 目录，不要只复制单个 exe。
  4. 推荐运行解压目录中的 install.cmd 安装到当前 Windows 用户目录，并创建桌面与开始菜单快捷方式。
  5. 安装器会关闭旧飞享 IM 进程，并替换指向旧 FeiXiangIM.exe 或 im_flutter.exe 的快捷方式。
  6. 也可以不安装，直接运行解压目录中的 FeiXiangIM.exe。
  7. 需要卸载时运行解压目录中的 uninstall.cmd。
